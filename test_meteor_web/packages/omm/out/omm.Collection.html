update(id, updateFunction)

Performs an update on an object in the collection. After the update the object is attempted to be saved to the collection. If the object has changed between the time it was loaded and the time it is saved, the whole process is repeated. This means that the updateFunction might be called more than once.
Parameters:
Name Type Description
id the id of the object
updateFunction the function that alters the loaded object
